470. In a computer system, which device is functionally opposite of a
Keyboard? (a) Trackball (b) Mouse (c) Joystick (d) Printer

The correct answer is:

**(d) Printer**

### 🧠 Explanation (simple)

- A **keyboard** is an **input device** (you send data *into* the
  computer)
- A **printer** is an **output device** (computer sends data *out* to
  paper)

👉 So they are **functionally opposite**

### ❌ Why others are wrong:

- **Trackball** → input device
- **Mouse** → input device
- **Joystick** → input device

### 🔑 One-line:

👉 **Keyboard = input, Printer = output → opposite** ✅

481. Batch processing was mainly used in which generation. (a) First (b)
Second (c) Third (d) Fourth

\
Yeah this question is confusing if you don't know "generations" 😄 ---
let's make it simple.

## 🧠 What the question means

It's asking:

👉 "In which **generation of computers** was **batch processing** mainly
used?"

## ⚙️ What is Batch Processing?

Batch processing =\
👉 **Jobs are collected and processed together (in batches), not
instantly**

Example:

- You submit 10 programs
- Computer runs them **one by one later**, without user interaction
