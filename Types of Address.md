# Types of Address

In an operating system, addresses are classified based on how memory is viewed and accessed.

## 1. Logical Address
- Generated by the CPU during program execution
- Also called **virtual address**
- Used by programs
- Must be translated before accessing memory

## 2. Physical Address
- Actual address in main memory (RAM)
- Used by the memory unit
- Obtained after address translation

## 3. Virtual Address
- Another name for logical address in modern systems
- Part of virtual memory system
- Allows programs to use more memory than physically available

## Summary
- Logical/Virtual Address → Used by CPU
- Physical Address → Used by Memory
