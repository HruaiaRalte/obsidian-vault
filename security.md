[]{#anchor}511. Which file system is most secure?

\(a\) FAT16 (b) FAT32 (c) NTFS (d) HFS

\
The correct answer is:

**(c) NTFS**

## 🧠 Explanation (simple)

**NTFS (New Technology File System)** is the most secure because it
supports:

- File/folder **permissions**
- **Encryption**
- **Access control**
- **File recovery features**

👉 That's why it's used in modern Windows systems ✅

## ❌ Why others are wrong:

- FAT16 / FAT32 → no security features
- HFS → used in Mac, but not as secure as NTFS (in this context)
