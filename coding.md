505\. Polymorphism refers to: (a) more than one form (b) inheritance (c)
data hiding (d) All of these

The correct answer is:

**(a) more than one form**

**Explanation:**\
**Polymorphism** in programming means **\"many forms\"**---the ability
of a function, method, or object to behave differently depending on
context (e.g., method overloading, method overriding).

It is related to object-oriented concepts but is **not the same as
inheritance or data hiding**, so option (d) is incorrect

The correct answer is:

**(a) Reusability**

**Explanation:**\
In **Object-Oriented Programming (OOP)**, **inheritance** allows a class
to reuse properties and methods of another class. This promotes **code
reusability**, reduces redundancy, and makes the system easier to
maintain.
