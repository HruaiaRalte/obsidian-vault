# Bits, Bytes, Words

## 🧠 Overview
Computers store and process all data using **binary (0s and 1s)**.  
These are organized into units called **bits, bytes, and words**.

---

## 🔹 Bit
- Smallest unit of data
- Can be:
  - `0` (off)
  - `1` (on)

👉 A single flip-flop stores **1 bit**
