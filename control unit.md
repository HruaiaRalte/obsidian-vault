554\. What is the control unit's function in the CPU? (a) transfer data
(b) store instructions (c) perform logic functions (d) decode
instructions

The correct answer is:

**(d) decode instructions**

### Explanation:

The **Control Unit (CU)** in the CPU is responsible for:

- Interpreting (decoding) instructions
- Directing other components (ALU, memory, registers)

Other options:

- \(a\) transfer data → handled by buses
- \(b\) store instructions → done by memory
- \(c\) perform logic functions → done by ALU

So, **Control Unit = decode + control operations** ✅

556\. What is the administrative section of the computer system ? (a)
Memory Unit (b) Input Unit (c) Control Unit (d) Central Processing Unit

The correct answer is:

**(c) Control Unit**

### Explanation:

The **Control Unit (CU)** is called the **administrative (managerial)
section** of the computer because it:

- Controls all operations of the system
- Directs data flow between components
- Coordinates CPU activities

Quick idea:

- **Control Unit = "Manager / Boss" of the computer** ✅

Other options:

- Memory Unit → stores data
- Input Unit → takes input
- CPU → includes CU + ALU (not just administrative part)

So, the best answer is **Control Unit** 👍
