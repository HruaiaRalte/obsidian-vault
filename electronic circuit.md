# ⚡ What is an **electronic circuit**?

An **electronic circuit** is a **path made of electronic components** that allows electricity to flow and do something useful.

---

# 🧠 Simple idea

> A circuit = a loop where electricity moves and performs a task

---

# 📦 Basic example

Think of a simple setup:

- Battery 🔋 → provides power
    
- Wire ➖ → carries electricity
    
- Bulb 💡 → uses electricity (lights up)
    

👉 When connected in a loop:

- Electricity flows
    
- Bulb turns ON
    

That loop = **circuit**

---

# 🔌 In computers

Instead of bulbs, circuits use tiny components like:

- Transistors
    
- Resistors
    
- Capacitors
    

👉 These control electricity very precisely

---

# ⚡ How this connects to flip-flops

A flip-flop is:

> a **special electronic circuit** designed to store a value (0 or 1)

Instead of lighting a bulb, it:

- stores ON (1)
    
- or OFF (0)
    

---

# 🧠 Analogy

- Circuit → roads
    
- Electricity → cars
    
- Components → traffic signals controlling flow
    

---

# 🔥 Why circuits matter

All digital systems are built from circuits:

- Flip-flops → store bits
    
- Registers → store multiple bits
    
- CPU → huge network of circuits
    

---

# 🧠 One-line summary

> An electronic circuit is a connected path that lets electricity flow and perform a function.
