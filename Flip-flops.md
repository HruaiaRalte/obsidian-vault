Flip-flops sound complicated, but they’re actually a simple idea 👇

---

# 🧠 What is a flip-flop?

A **flip-flop** is a tiny [[electronic circuit]] that can **remember one value**:

👉 either **0 or 1**

So:

> **Flip-flop = smallest memory unit in digital electronics**

---

# ⚡ Why is it called “flip-flop”?

Because it can:

- **flip** → change from 0 → 1
    
- **flop** → change from 1 → 0
    

…and then **stay in that state** until told to change again.

---

# 📦 Simple analogy

Think of it like a **light switch**:

- OFF → `0`
    
- ON → `1`
    

👉 It stays ON or OFF until you flip it again.

---

# 🔌 How it works (conceptually)

A flip-flop:

- receives input signals
    
- changes state based on those inputs
    
- **stores that state** even after input is gone
    

---

# 🧩 Types of flip-flops (just for idea)

- **SR flip-flop** → basic set/reset
    
- **D flip-flop** → stores input value (most common)
    
- **JK flip-flop** → improved version
    
- **T flip-flop** → toggles state
    

👉 You don’t need all this now—just know they store bits

---

# 🔗 Why they matter

Flip-flops are used to build:

- **Registers** (like you saw)
    
- Counters
    
- Memory systems
    
- Parts of CPUs
    

---

# 🧠 Simple summary

> A flip-flop is a tiny circuit that can **store one bit (0 or 1)** and remember it until changed.
