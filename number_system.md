490. Hexadecimal number system has:

\(a\) 8 symbols (b) 16 symbols (c) 32 symbols (d) 64 symbols\
\
\
The correct answer is:

**(b) 16 symbols**

### 🧠 Explanation (simple)

The **hexadecimal number system** uses **16 symbols**:

👉 **0--9** (10 digits)\
👉 **A--F** (6 letters)

So total = **10 + 6 = 16**

### 💡 Example:

- A = 10
- B = 11
- C = 12
- D = 13
- E = 14
- F = 15
