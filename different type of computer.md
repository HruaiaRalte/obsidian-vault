[]{#anchor}400. In analog computer

\(a\) Input is first converted (b) Input is never converted (c) Output
is digital (d) All of the above

\
The correct answer is:

👉 **(b) Input is never converted**

### 💡 Explanation:

An **analog computer** works with **continuous data** (like voltage,
temperature, speed).

- It **does not convert input into digital form**
- It processes data directly in analog form ✅

### ❌ Why others are wrong:

- **Input is first converted** → happens in digital systems
- **Output is digital** → analog computers produce analog output
- **All of the above** → incorrect

✔️ Final answer: **Input is never converted**
