# RAM --- BASIC DEFINITION

**RAM (Random Access Memory)** is the **main memory of a computer used
to temporarily store programs and data while the CPU is running them.**

Key properties:

  Feature         Description
  --------------- ---------------------------------------
  Volatile        Data lost when power off
  Random access   Any location can be accessed directly
  Fast            Faster than secondary storage

# TYPES OF RAM

## 1️⃣ SRAM (Static RAM)

Characteristics:

- uses **flip-flop circuits**
- **no refreshing needed**
- **very fast**
- **expensive**

Used for:

    CPU cache memory

## 2️⃣ DRAM (Dynamic RAM)

Characteristics:

- uses **capacitor + transistor**
- **needs refreshing**
- **high density**
- **cheaper**

Used for:

    main memory

# DRAM GENERATIONS

  Type       Description
  ---------- -----------------------
  FPM DRAM   Fast Page Mode
  EDO DRAM   Extended Data Output
  SDRAM      Synchronous DRAM
  DDR        Double Data Rate
  DDR2       Faster DDR
  DDR3       Improved speed
  DDR4       Modern RAM
  DDR5       Latest high-speed RAM

# MEMORY PERFORMANCE TERMS

  ------------- -------------------------
  Access Time   Time to retrieve data

  Latency       Delay before transfer
  Bandwidth     Data transfer capacity

# MEMORY MODULE TYPES

  SIMM      Older computers

  ECC RAM   Servers

    Registers

## 🧠 Explanation (simple)

**Dynamic RAM (DRAM)**:

- Stores data using **capacitors**
- Charge leaks quickly
- So it must be **refreshed many times per second** ✅

## ❌ Why others are wrong:

- EPROM → non-volatile, no refresh needed
- ROM → permanent memory
- Static RAM (SRAM) → does NOT need refresh
