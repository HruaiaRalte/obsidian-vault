# WHO CREATED THE WORLD WIDE WEB

The **World Wide Web** was invented by **Tim Berners-Lee** in **1989**
at **CERN**.

He also created:

- **HTML**
- **HTTP**
- **URL**

These are the **3 core technologies of the web**.

# CORE WWW TECHNOLOGIES

    WWW Core Technologies
    │
    ├─ HTML → Web page structure
    ├─ HTTP → Communication protocol
    └─ URL → Address of web resources

# INTERNET vs WWW

  Internet         WWW
  ---------------- ------------------------
  Global network   Web information system
  Infrastructure   Service on internet
  Uses TCP/IP      Uses HTTP

# FOUR THINGS NEEDED TO CONNECT INTERNET USING TELEPHONE LINE

1️⃣ **Computer**\
Device used to access the Internet.

2️⃣ **Modem**\
Device that converts **digital signals ↔ analog signals**.

3️⃣ **Telephone Line**\
Communication medium used for dial-up connection.

4️⃣ **Internet Service Provider (ISP)**\
Company that provides Internet access.

Examples of ISPs:

- Airtel
- Jio
- BSNL
