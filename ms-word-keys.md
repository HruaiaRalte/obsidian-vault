Here is a comprehensive list of the most essential **Microsoft Word**
shortcut keys, categorized by their function. These are standard for
Windows versions of Word.

### Basic Document Operations

These are the fundamental shortcuts for managing your files.

  ****Shortcut****   ****Function****
  ------------------ ----------------------------------
  Ctrl + N           Create a **new** document.
  Ctrl + O           **Open** an existing document.
  Ctrl + S           **Save** the current document.
  Ctrl + W           **Close** the document.
  Ctrl + P           **Print** the document.
  F12                Open the **Save As** dialog box.

### Text Editing & Formatting

Use these to change how your text looks and to move content around.

  ****Shortcut****   ****Function****
  ------------------ ----------------------------------------
  Ctrl + B           Apply **Bold** formatting.
  Ctrl + I           Apply *Italic* formatting.
  Ctrl + U           Apply \<u\>Underline\</u\> formatting.
  Ctrl + C           **Copy** selected text or object.
  Ctrl + X           **Cut** selected text or object.
  Ctrl + V           **Paste** the copied/cut content.
  Ctrl + Z           **Undo** the last action.
  Ctrl + Y           **Redo** the last action.
  Ctrl + K           Insert a **Hyperlink**.

### Paragraph Alignment

These shortcuts help you position your text on the page quickly.

  ****Shortcut****   ****Function****
  ------------------ -----------------------------------------------------------
  Ctrl + L           **Align Left** (standard for body text).
  Ctrl + E           **Center** Align (common for titles).
  Ctrl + R           **Align Right**.
  Ctrl + J           **Justify** (aligns text to both left and right margins).
  Ctrl + M           **Indent** the paragraph from the left.

### Navigation & Selection

Quickly move through your document without using the mouse.

  ****Shortcut****   ****Function****
  ------------------ -------------------------------------------------------
  Ctrl + A           **Select All** content in the document.
  Ctrl + F           **Find** specific text.
  Ctrl + H           **Replace** text.
  Ctrl + G           **Go To** a specific page or section.
  Ctrl + Home        Move the cursor to the **beginning** of the document.
  Ctrl + End         Move the cursor to the **end** of the document.
  Ctrl + Enter       Insert a **Page Break**.

### Font Size Control

A quick way to resize text without opening the font menu.

  ****Shortcut****    ****Function****
  ------------------- -----------------------------------------------------
  Ctrl + \[           **Decrease** font size by 1 point.
  Ctrl + \]           **Increase** font size by 1 point.
  Ctrl + Shift + \>   **Increase** font size to the next preset size.
  Ctrl + Shift + \<   **Decrease** font size to the previous preset size.

\
MS Word Function Keys (F1--F12)

::: {}
The function keys are \"power user\" shortcuts that perform specific
commands without needing the **Ctrl** or **Alt** keys.

  ****Key****   ****Function****
  ------------- ------------------------------------------------------------------------------------------------------
  F1            Open **Help** or the Office Assistant.
  F2            **Move** text or graphics (Select text, press F2, then click where you want it and press Enter).
  F4            **Repeat** your last action (very useful for repetitive formatting).
  F5            Open the **Go To** tab (also includes Find and Replace).
  F7            Launch **Spelling and Grammar** check.
  F8            **Extend Selection** (Pressing it multiple times selects a word, then a sentence, then a paragraph).
  F9            **Update** selected fields (like a Table of Contents or a formula).
  F10           Show **Key Tips** (labels for the Ribbon menu).
  F11           Go to the **Next Field**.
  F12           **Save As** (opens the dialog box instantly).

### Pro-Tip: The \"Shift + Function\" Combos

You can get even more out of these keys by holding **Shift**:

- **Shift + F3:** Change the **Case** of selected text (UPPERCASE,
  lowercase, or Title Case).
- **Shift + F5:** Jump back to the **last place you edited**.
- **Shift + F7:** Open the **Thesaurus** for the selected word.

For exams like the **LDC (Lower Division Clerk)**, the syllabus is
designed to test your efficiency in an office environment. The focus is
usually on how well you can handle documents, manage data, and
communicate through official letters.

::: {}
Here are the key categories you should focus on, broken down by how they
appear in the exam:

### 1. Microsoft Word (Word Processing)

This is the most important section for an LDC. You need to know how to
create and format official documents.

- **Formatting:** Bold, Italics, Underline, Subscript/Superscript
  (\$X\^2\$, \$H_2O\$), and Change Case.
- **Page Layout:** Setting margins, page orientation (Portrait vs.
  Landscape), and paper size (A4, Letter).
- **References:** How to insert Footnotes, Endnotes, and Table of
  Contents.
- **Mail Merge:** This is a common \"advanced\" question---know how to
  send one letter to multiple recipients.
- **Shortcuts:** Memorize the **Ctrl** and **Function (F1-F12)** keys we
  discussed earlier.

### 2. Microsoft Excel (Spreadsheets)

Since LDCs handle data and payroll, you must understand the basics of
rows, columns, and cells.

- **Basic Formulas:** `=SUM()`, `=AVERAGE()`, `=MIN()`, `=MAX()`, and
  `=COUNT()`.
- **Data Tools:** Sorting (A to Z), Filtering, and Data Validation.
- **Cell Referencing:** The difference between Relative (A1) and
  Absolute (\$A\$1) references.
- **Charts:** Knowing which chart is best for what data (e.g., Pie
  charts for percentages).

### 3. Computer Fundamentals & OS

These are the \"theory\" questions about how a computer works.

- **Operating Systems:** Understanding multitasking (Windows/Linux) vs.
  single-tasking (DOS).
- **Hardware:** Input devices (Keyboard, Scanner) vs. Output devices
  (Printer, Monitor).
- **Storage Units:** Bits, Bytes, KB, MB, GB, TB, and PB.
- **Memory:** Difference between RAM (Volatile) and ROM (Non-volatile).

### 4. Internet & Communication

- **Web Browsing:** Terms like URL, HTTP/HTTPS, IP Address, and DNS.
- **Email Etiquette:** Understanding **CC** (Carbon Copy) vs. **BCC**
  (Blind Carbon Copy).
- **Cyber Security:** Basic knowledge of viruses, malware, and
  firewalls.
