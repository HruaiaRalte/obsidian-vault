# Addressing Concepts

Addressing concepts define how memory locations are identified and accessed in a computer system.

## 1. Address Bus
- A set of wires used to carry memory addresses
- Determines how many memory locations can be accessed
- Size of address bus (n bits) → can address 2^n locations

## 2. Address Range
- The range of addresses available in a system
- Depends on address bus size
- Example: 32-bit system → 0 to 2^32 - 1

## 3. Address Space (Overview)
- The set of all possible addresses a program can use
- Includes logical (virtual) and physical address space
- Each process has its own address space

## Key Idea
- These concepts define **how memory is structured and accessed efficiently**
