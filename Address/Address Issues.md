

Address issues arise when there are problems in accessing or managing memory addresses.

## 1. Invalid Address
- Occurs when a program tries to access a memory location that is not allowed
- Can cause program crashes or exceptions

## 2. Protection Fault
- Happens when a process accesses memory without proper permissions
- Example: writing to a read-only memory location

## 3. Fragmentation
- Inefficient use of memory due to allocation methods

### Types of Fragmentation
- Internal Fragmentation
  - Unused space inside allocated memory block
- External Fragmentation
  - Free memory is scattered in small blocks

## Key Idea
- Address issues affect **system stability, performance, and memory efficiency**
