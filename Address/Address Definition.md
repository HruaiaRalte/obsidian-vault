# 

An **address** in an operating system is a unique identifier used to specify a particular location in memory.

It tells the computer **where data or instructions are stored** so that the CPU can access them.

## Key Points
- Represents a location in memory
- Used by the CPU to read/write data
- Can be in different forms (logical, physical, virtual)

## Example
If a value is stored in memory, its address is like its **home location**, allowing the system to find it quickly.
