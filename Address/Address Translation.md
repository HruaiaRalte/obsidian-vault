# Address Translation

Address translation is the process of converting a **logical address (generated by the CPU)** into a **physical address (used by main memory)**.

## How It Works
- CPU generates a logical (virtual) address
- This address is sent to the MMU (Memory Management Unit)
- MMU translates it into a physical address
- The physical address is used to access memory

## Key Components
- Logical Address
- Physical Address
- MMU (Memory Management Unit)

## Why It Is Needed
- Programs use logical addresses for flexibility
- Physical memory locations may change
- Enables virtual memory and process isolation

## Example
Logical Address → MMU → Physical Address → Memory Access
