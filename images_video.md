# 1. IMAGE

## 1.1 Definition

An **image** is a **digital representation of a picture stored in a
computer using pixels**.

    Image
     ├─ Types of images
     ├─ Image formats
     ├─ Color models
     ├─ Image resolution
     ├─ Compression
     └─ Image editing

# 1.2 Types of Images

    Image Types
     ├─ Raster Image (Bitmap)
     └─ Vector Image

### Raster Image

- Made of **pixels**
- Quality depends on **resolution**
- Used for photographs

Examples:

- JPG
- PNG
- BMP

### Vector Image

- Made of **mathematical shapes**
- Can scale without losing quality

Examples:

- SVG
- AI
- EPS

# 1.3 Image File Formats

  Format           Full Form                          Created by              Notes
  ---------------- ---------------------------------- ----------------------- --------------------------
  **JPEG / JPG**   Joint Photographic Experts Group   JPEG committee          Most common photo format
  **PNG**          Portable Network Graphics          PNG Development Group   Supports transparency
  **GIF**          Graphics Interchange Format        CompuServe              Supports animation
  **BMP**          Bitmap                             Microsoft               Large file size
  **TIFF**         Tagged Image File Format           Aldus Corporation       High-quality images
  **SVG**          Scalable Vector Graphics           W3C                     Vector format

# 1.4 Image Resolution

Resolution = **number of pixels in an image**

Example:

    1920 × 1080 pixels

Higher resolution → clearer image.

Measured in:

- **PPI** -- Pixels per inch
- **DPI** -- Dots per inch

# 1.5 Color Models

    Color Models
     ├─ RGB
     ├─ CMYK
     └─ Grayscale

### RGB

Full form: **Red Green Blue**\
Used for **screens and digital displays**

### CMYK

Full form: **Cyan Magenta Yellow Black**\
Used for **printing**

# 1.6 Image Compression

Purpose: **reduce file size**

Types:

    Compression
     ├─ Lossy
     └─ Lossless

Lossy

- Some quality lost
- Example: **JPEG**

Lossless

- No quality loss
- Example: **PNG**

# 1.7 Image Editing Software

Examples:

- Adobe Photoshop
- GIMP
- CorelDRAW
- Paint

# 2. VIDEO

## 2.1 Definition

A **video** is a **sequence of images (frames) displayed rapidly to
create motion**.

Example:

    1 second video ≈ 24–60 frames

# 2.2 Video Structure

    Video
     ├─ Frames
     ├─ Frame rate
     ├─ Resolution
     ├─ Bitrate
     └─ Compression

# 2.3 Frame Rate

Frame rate = **number of frames shown per second**

Common frame rates:

  FPS      Use
  -------- ----------------------
  24 fps   Movies
  30 fps   Television
  60 fps   Games / smooth video

# 2.4 Video Resolution

  Resolution   Size
  ------------ -----------
  480p         SD
  720p         HD
  1080p        Full HD
  4K           Ultra HD
  8K           Super UHD

# 2.5 Video Formats

  Format    Full Form                Developed by       Notes
  --------- ------------------------ ------------------ -------------------------
  **MP4**   MPEG-4 Part 14           MPEG               Most common
  **AVI**   Audio Video Interleave   Microsoft          Older format
  **MKV**   Matroska Video           Matroska Project   Open source
  **MOV**   QuickTime Movie          Apple              Used in QuickTime
  **WMV**   Windows Media Video      Microsoft          Windows format
  **FLV**   Flash Video              Adobe              Used in older web video

# 2.6 Video Codecs

Codec = **compressor/decompressor**

Examples:

- H.264
- H.265 (HEVC)
- VP9
- AV1

# 2.7 Video Compression

Purpose:

- Reduce file size
- Enable streaming

Types:

    Video Compression
     ├─ Lossy compression
     └─ Lossless compression

# 2.8 Video Editing Software

Examples:

- Adobe Premiere Pro
- Final Cut Pro
- DaVinci Resolve
- OpenShot

# 3. Relationship Between Image and Video

    Video
       ↓
    Collection of many images
       ↓
    Images shown rapidly
       ↓
    Creates motion

Example:

    1 second video
    = 30 images

# 4. Exam Important Points

Image

- Made of **pixels**
- Stored as **image formats**

Video

- Made of **frames**
- Stored as **video formats**

Main difference:

  Image            Video
  ---------------- ----------------------
  Single picture   Sequence of pictures
  No motion        Motion
