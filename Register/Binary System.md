---

Binary System

Overview  
The binary system is a number system used by computers that uses only two digits: 0 and 1.  
It is called base-2 because it has two possible values.

---

How it works  
In binary, each position represents a power of 2 (instead of 10 like in decimal).

Example:  
Binary: 1011

Breakdown:  
1 × 2³ = 8  
0 × 2² = 0  
1 × 2¹ = 2  
1 × 2⁰ = 1

Total = 8 + 0 + 2 + 1 = 11 (decimal)

---

Place values (right to left)  
2⁰ = 1  
2¹ = 2  
2² = 4  
2³ = 8  
2⁴ = 16  
2⁵ = 32

---

Why computers use binary

- Electronic circuits have two states: ON and OFF
    
- ON → 1
    
- OFF → 0
    
- This makes binary reliable and easy to implement in hardware
    

---

Examples

Decimal → Binary  
0 → 0  
1 → 1  
2 → 10  
3 → 11  
4 → 100  
5 → 101  
10 → 1010

---

Binary to decimal example

Binary: 1101

1 × 8 = 8  
1 × 4 = 4  
0 × 2 = 0  
1 × 1 = 1

Total = 13

---

Where it is used

- Data storage
    
- CPU operations
    
- Memory (RAM, registers)
    
- Machine instructions
    

---

Summary

- Binary uses only 0 and 1
    
- It is base-2
    
- Every number can be represented using powers of 2
    
- It is the foundation of all computing
