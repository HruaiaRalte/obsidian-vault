# Bits, Bytes, Words

## 🧠 Overview
Computers store and process all data using **binary (0s and 1s)**.  
These are organized into units called **bits, bytes, and words**.

---

## 🔹 Bit
- Smallest unit of data
- Can be:
  - `0` (off)
  - `1` (on)

👉 A single [[flip-flop]] stores **1 bit**

---

## 🔹 Byte
- 1 Byte = **8 bits**

Example:



👉 A byte can represent:
- A character (like `A`, `1`, `@`)
- Numbers from **0 to 255**

---

## 🔹 Word
- A **word** is a group of bytes
- Size depends on CPU architecture

Common sizes:
- 16-bit (2 bytes)
- 32-bit (4 bytes)
- 64-bit (8 bytes)

👉 Modern CPUs usually use **64-bit words**

---

## 🔗 Relationship

---

## ⚡ Why this matters

- Registers store data in **bits/bytes/words**
- CPU processes data in **word size**
- Memory is measured in bytes (KB, MB, GB)

---

## 🧠 Summary

- Bit → smallest unit (0 or 1)  
- Byte → 8 bits  
- Word → CPU-sized data unit  

👉 Everything in a computer is built from these
