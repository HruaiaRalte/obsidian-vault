**(a) con**

**CON** cannot be used as a folder name in Windows.

- **CON** is a **reserved word** in Windows operating system
- Windows has several **reserved device names** that cannot be used as
  file or folder names
- If you try to create a folder named \"CON\", Windows will **refuse or
  rename it**

**Windows Reserved Names (cannot be used as folder/file names):**

    CON → Console (standard input/output)
    PRN → Printer
    AUX → Auxiliary device
    NUL → Null device
    COM1, COM2, COM3... → Serial ports
    LPT1, LPT2, LPT3... → Parallel ports

**Why not the others?**

- **(b) can** --- valid folder name ✓
- **(c) cam** --- valid folder name ✓
- **(d) dam** --- valid folder name ✓

**Other characters also invalid in Windows folder names:**

    \ / : * ? " < > |

These special characters are **not allowed** in any Windows file or
folder name.

**CON** is the most famous reserved name in Windows --- it is
historically linked to the **console device** of the operating system.

**(b) OMR**

**OMR (Optical Mark Recognition/Reader)** is mainly used for checking
objective type answer papers.

- OMR reads **pencil or pen marks** made in predetermined positions
- It detects **filled bubbles/circles** on specially designed sheets
- It is widely used for **checking MCQ/objective type answer sheets**
- It is **fast, accurate, and efficient** for bulk checking

**Common uses of OMR:**

    ✓ Competitive exam answer sheets
    ✓ Multiple choice question papers
    ✓ Survey forms
    ✓ Lottery tickets
    ✓ Attendance sheets
    ✓ Election ballots

**Why not the others?**

- **(a) MICR (Magnetic Ink Character Recognition)** --- used for reading
  **bank cheques** and financial documents
- **(c) OCR (Optical Character Recognition)** --- used for reading
  **printed or handwritten text** and converting to digital format

**Comparison of all three:**

    OMR → Reads marks/bubbles
           (Exam answer sheets)

    MICR → Reads magnetic ink characters
           (Bank cheques)

    OCR → Reads printed/handwritten text
           (Document scanning)

**Examples of OMR usage in India:**

    ✓ UPSC exam sheets
    ✓ SSC exam sheets
    ✓ NEET answer sheets
    ✓ LDC exam sheets
    ✓ Banking exam sheets

OMR is the **fastest and most reliable** method for checking large
volumes of objective type answer papers.

**(c) Codewords**

**Codewords** (Passwords) are coded entries used to gain access to a
computer system.

- Codewords are **secret coded entries** that authenticate a user
- They are used to **verify identity** before granting access
- They **protect unauthorized access** to computer systems
- Also commonly known as **passwords**

**Why not the others?**

- **(a) Security commands** --- these are **instructions** given to
  security software, not access codes
- **(b) Entry codes** --- not a standard computer terminology
- **(d) Mnemonics** --- these are **memory aids** used in assembly
  language programming (like MOV, ADD, SUB)

**Types of Codewords/Access entries:**

    Password       → Text based secret code
    PIN            → Numeric code
    Passphrase     → Long phrase as password
    OTP            → One Time Password
    Biometric      → Fingerprint/Face ID
    Token          → Hardware security key

**Properties of a good Codeword/Password:**

    ✓ Minimum 8 characters
    ✓ Mix of uppercase and lowercase
    ✓ Include numbers
    ✓ Include special characters
    ✓ Not easily guessable
    ✓ Changed regularly
    ✗ Never share with others
    ✗ Never use personal info

Codewords are the most fundamental form of **computer security** used to
protect systems from unauthorized access.

**(d) DSL**

**DSL (Digital Subscriber Line)** is the technology used to provide
internet by transferring data over wires of telephone network.

- DSL provides **high speed internet** through existing **telephone
  copper wires**
- It allows **simultaneous use** of internet and telephone on same line
- It is widely used for **broadband internet** connections
- It transfers data in **digital format** over telephone lines

**Types of DSL:**

    ADSL → Asymmetric DSL
            (faster download than upload)
            Most common for home use

    SDSL → Symmetric DSL
            (equal upload and download)
            Used for businesses

    VDSL → Very high speed DSL
            (fastest DSL type)

    HDSL → High data rate DSL
            (used for T1/E1 lines)

    ADSL2 → Improved ADSL
            (longer range, faster speed)

**Why not the others?**

- **(a) Transmitter** --- a device that **sends signals**, not an
  internet technology
- **(b) Diodes** --- electronic **components**, not internet technology
- **(c) HHL** --- not a standard internet technology term

**Internet connection technologies comparison:**

    DSL     → Telephone copper wires
    Cable   → TV cable network
    Fiber   → Optical fiber cables
    Dial up → Old telephone modem
    Satellite→ Space satellites
    WiFi    → Wireless radio waves
    WiMAX  → Wide area wireless

**DSL Speed ranges:**

    ADSL   → Up to 8 Mbps download
    ADSL2  → Up to 24 Mbps download
    VDSL   → Up to 52 Mbps download
    VDSL2  → Up to 100 Mbps download

DSL revolutionized internet access by utilizing the **existing telephone
infrastructure** to provide broadband internet connectivity.

**(d) ISDN**

**ISDN (Integrated Services Digital Network)** is a set of communication
standards for digital transmission of voice and video.

- ISDN provides **simultaneous digital transmission** of voice, video,
  and data
- It uses **existing telephone network** infrastructure
- It offers **faster and more reliable** communication than analog
  systems
- It is a **circuit switched** telephone network system

**Full form breakdown:**

    I → Integrated (combines multiple services)
    S → Services   (voice, video, data)
    D → Digital    (digital transmission)
    N → Network    (communication network)

**Why not the others?**

- **(a) NIC (Network Interface Card)** --- hardware that **connects
  computer to network**
- **(b) PSTN (Public Switched Telephone Network)** --- traditional
  **analog telephone network**
- **(c) VAN (Value Added Network)** --- private network providing
  **extra services** beyond data transfer

**ISDN Channels:**

    B Channel → Bearer channel
                64 Kbps
                carries voice/data

    D Channel → Delta channel
                16 Kbps
                carries signaling info

**ISDN Types:**

    BRI (Basic Rate Interface)
    → 2B + 1D channels
    → Used for homes/small businesses
    → 128 Kbps total speed

    PRI (Primary Rate Interface)
    → 23B + 1D (USA/Japan)
    → 30B + 1D (Europe/India)
    → Used for large businesses

**Communication Standards comparison:**

    ISDN → Digital voice/video/data
    DSL  → Internet over phone lines
    PSTN → Analog telephone network
    VoIP → Voice over Internet Protocol
    GSM  → Mobile communication standard

ISDN was a **revolutionary technology** that enabled digital
communication over existing telephone infrastructure before being
largely replaced by **DSL and fiber optic** technologies.

- The standard Windows keyboard has **104 keys**
- It includes **Windows specific keys** like Windows key and Menu key
- It is the most commonly used keyboard layout today

**Evolution of Keyboard keys:**

    Original PC Keyboard → 84 keys
                            (IBM 1981)

    AT Keyboard          → 84 keys
                            (IBM 1984)

    Enhanced Keyboard    → 101 keys
                            (IBM 1986)

    Windows Keyboard     → 104 keys ✓
                            (Microsoft 1994)
                            Added 3 extra keys:
                            → 2 Windows keys
                            → 1 Menu/Application key

    Extended Keyboard    → 109 keys
                            (Japanese layout)

**104 Key breakdown:**

    Alphanumeric keys → 26 letters
                         10 numbers
                         Special characters

    Function keys     → F1 to F12 (12 keys)

    Control keys      → Ctrl, Alt, Shift
                         Caps Lock, Tab
                         Enter, Backspace

    Navigation keys   → Arrow keys
                         Home, End
                         Page Up, Page Down
                         Insert, Delete

    Numeric Keypad    → Numbers 0-9
                         Mathematical operators
                         Num Lock

    Windows keys      → 2 Windows keys (⊞)
                         1 Menu key

**Special Windows keys added in 104 keyboard:**

    Windows Key (⊞) → Opens Start Menu
                       Keyboard shortcuts

    Menu Key        → Opens right click
                       context menu

**Keyboard types comparison:**

    84 keys → Original PC keyboard
    101 keys → Enhanced keyboard
    104 keys → Standard Windows keyboard ✓
    109 keys → Extended/Japanese keyboard

The **104 key Windows keyboard** is the most widely used keyboard layout
