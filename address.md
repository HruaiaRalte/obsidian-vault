392\. In operating system, each process has its own

\(a\) address space (b) open files (c) pending alarms (d) all of
theseAddress

The correct answer is:

👉 **(d) All of these**

### 💡 Explanation:

In an **operating system**, each process has its own:

- **Address space** → its own memory area
- **Open files** → files it is currently using
- **Pending alarms** → scheduled signals/timers

✔️ Final answer: **All of these**
