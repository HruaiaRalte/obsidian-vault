489\. Built-in set of machine code instructions of computer are called:

\(a\) instruction set (b) transfer of data (c) logical operations (d)
logical set

Instruction set

The correct answer is:

**(a) Instruction set**

### 🧠 Explanation (simple)

A computer has a built-in collection of machine-level commands called
the **instruction set**.

👉 These are the basic operations the CPU understands, like:

- Add
- Subtract
- Move data
- Compare

### 💡 Example

The CPU follows instructions like:

- ADD
- LOAD
- STORE

All of these together = **Instruction Set**

### ❌ Why others are wrong:

- Transfer of data → just one type of operation
- Logical operations → only part of instructions
- Logical set → not a standard term

### 🔑 One-line:

👉 **Instruction set = all machine-level commands a CPU can execute** ✅

504\. A machine language instruction format consists of (a) Operand (b)
Operation code (c) Operation code and operand (d) None of the above

The correct answer is:

**(c) Operation code and operand**

**Explanation:**\
A machine language instruction typically consists of:

- **Operation code (opcode):** Specifies the action to be performed
- **Operand:** Specifies the data or memory location on which the
  operation is performed

Together, they form the complete instruction format.
