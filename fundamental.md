520\. Which of the following extensions suggest that the file is a
backup copy? (a) Bak (b) Bas (c) Com (d) Txt

The correct answer is:

**(a) Bak**

## 🧠 Explanation (simple)

The **.bak** extension is used for:\
👉 **Backup files** (copy of original file)

Example:

- `file.txt` → original
- `file.bak` → backup copy

## ❌ Why others are wrong:

- Bas → BASIC program file
- Com → executable file
- Txt → text file

## 📚 Topic Classification

👉 **Computer Fundamentals → File System **o**→ File Extensions**

### 🔑 One-line:

👉 **.bak = backup file** ✅

553\. IC chips used in computers are usually made of (a) Lead (b)
Silicon (c) Chromium (d) Gold

The correct answer is:

**(b) Silicon**

## 🧠 Explanation (simple)

**Integrated Circuits (IC chips)** are made from:\
👉 **Silicon**, a semiconductor material

- Conducts electricity under certain conditions
- Perfect for making electronic components

## ❌ Why others are wrong:

- Lead → not used for IC chips
- Chromium → not standard
- Gold → used for connectors, not chips

## 📚 Topic Classification

👉 **Computer Fundamentals → Electronic Components → IC Technology**

### 🔑 One-line:

👉 **IC chips are made of silicon** ✅
